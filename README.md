# 📥 دانلودر خودکار فایل‌ها + مرورگر وب برای GitHub

> ⚠️ **توجه مهم:** این ابزار فقط **لینک مستقیم** (Direct Link) رو پشتیبانی می‌کنه. یعنی لینکی که با زدن روی آن، فایل بلافاصله دانلود بشه.

## 🔧 تنظیمات اولیه

1. روی **Fork** بزنید
2. با نام دلخواه **Create Fork** را انتخاب کنید
3. به **Settings** → **Actions** → **General** برید
4. زیر **Workflow permissions** گزینه **Read and write permissions** رو انتخاب کنید
5. ذخیره کنید ✅

## 🎯 نحوه استفاده

فایل `README.md` رو باز کنید، **یک خط خالی** به انتهای آن اضافه کنید و کامیت کنید با این پیام:

### دانلود فایل:
```
download: https://example.com/file.zip
```

### بازدید و ذخیره صفحه وب:
```
visit: https://example.com/page
```


## 📂 فایل‌ها کجا هستند؟

- **دانلود فایل‌ها** → پوشه `downloads/`
- **صفحات وب** → پو  `pages/` (همراه اسکرین‌شات و تمام رسانه‌ها)
